# tom.ai

A one-page AI landing site with a coding-inspired dark aesthetic.

**Live:** https://mcws26.github.io/tom-ai/

## Features
- Hero with per-letter 3D type-in animation and blinking terminal caret
- Pinned scrollytelling benefits section with tilted, alternating cards
- Gooey liquid neon border effect on cards (SVG filter goo)
- Scroll-linked card numbering beneath the pinned title
- Scroll-triggered background crossfade from dark to purple→cyan gradient
- Typed code background in the signup section with cursor spotlight reveal
- Custom 30px black cursor with mix-blend inversion
- Lenis smooth scroll

## Stack
Single static `index.html` — vanilla HTML/CSS/JS, Lenis via CDN, Google Fonts (Space Grotesk + JetBrains Mono).
